Margin

Agrega un espacio a los lados de un elemento.

Si solo se especifica una medida, se aplica a todos los lados.

Si se especifican dos numeros: se aplican vertical y horizontalmente en ese orden.

Si se especifican cuatro numeros: se aplican a cada lado, comenzando por arriba y yendo en direccion del reloj.

Padding

Define el espacio entre el borde de un elemento y su contenido.

La cantidad de distancia a cada lado en base a los numeros dados es la misma que con margin.

Display

Sirve para determinar cómo se ordenan las cajas que se utilizan para un elemento. Esto puede ser varios en una linea, uno por linea, etc. Hay dos tipos principales:
inline: Ordenar varios contenedores juntos, en una sola linea.
bloque: Ordear los elementos que contiene detro de una linea. Esto puede ser solo un contenedor o varios inline. El texto colorido en la seccion "Color" tiene un "inline" display para evitar la nueva linea con cada palabra, ya que el tag "p" tiene display de bloque automaticamente.

Overflow

Sirve para definir la conducta de un elemento cuando es demasiado grande para caber dentro de la ventana. Puede ser que el contenido excesivo se vuelva invisible, que aparezca un scrollbar o que pase a otra línea en el caso del texto.

Position

Sirve para definir la posición de un elemento. Esto puede ser la posición normal (static), relativamente con esta posición normal (relative), fija a la página (fixed) o relativa a su elemento ancestro más cercano (absolute).
