div - Divisiones

Sirve para delimitar un grupo de elementos juntos, donde se le pueden agregar elementos estilísticos a todos los elementos que contiene usando css. Este texto está dentro de un div junto con el siguiente párrafo de texto.

header - Header

Sirve para delimitar un grupo de elementos, como con Div, pero especializado como una introducción para los siguientes elementos. Usualmente contiene ayuda para la navegación o listados.

span - Span

Span es para representar a los elementos que tiene dentro. Si se le cambia algo a span, se aplica a los elementos que tiene dentro.

article - Article

Articles sirven como una sección completamente encapsulada. No tiene repercusiones en otras partes, por lo que es transferible y modificable completamente dentro de sí misma. Este texto está dentro de un article.

section - Sección

Sirve para delimitar una sección genérica de un documento o aplicación. Es una agrupación temática de contenido, por lo que section es otro contenedor.

Este segundo párrafo se encuentra dentro de la misma sección que el párrafo pasado.

strong - Strong

Sirve para darle prioridad a su contenido. Sin embargo, no es un contenedor como div porque tiene ciertas restricciones en los elementos que puede contener. Este cuadro de texto está dentro de un strong, por lo que aparece en letras negritas.

em - Enfasis

Es como strong: sirve para ponerle enfasis a sus contenidos, pero tiene ciertas restricciones que no lo hacen un contenedor como tal. Este texto está dentro de un elemento em.

aside - Contenido secundario

Sirve para poner elementos que son secundarios, pero relacionados al contenido de alrededor. Normalmente se utiliza para poner sidebars o cosas similares. Este texto y el subtítulo relacionado están dentro de un elemento aside.

nav - Links de navegación

Una sección con links de navegación que llevan a otras páginas o a otra parte de la misma página. Este texto y su subtítulo relacionado están dentro de una sección de navegación.

Listas

ul - Listas sin orden

Los elementos en las listas desordenadas no tienen un orden determinado. Por esto se representan con bullet points. Cada elemento en una lista se define con el tag "li".

  • 1
  • 5
  • 7
  • 33
  • 2
  • 9

ol - Listas ordenadas

Los elementos en las listas ordenadas ya tienen un orden determinado. Por esto se representan numerados.

  1. 1
  2. 2
  3. 5
  4. 7
  5. 9
  6. 33

table - Tablas

Tambien se pueden hacer tablas. Se agrega con una linea de headers hasta arriba con el tag thead.

El tag tr representa una linea en una tabla, mientras que td representa un elemento dentro de esa linea.

img - Imagenes

Sirven para poner un elemento visual en la pagina, com el que se ve abajo. Tienen un tipo de tag diferente, en el que se agregan los atributos (como parametros). "alt" es lo que sale cuando no se puede cargar la imagen. "src" es la funete de la imagen relativa a este archivo.

a - Links

Se pueden agregar hypelinks con el tag "a", los cuales hacen un enlace hacia otro recurso web. Se puede definir si se abre en una pesatana nueva o en la misma con el tag "Target". Un ejemplo de un hyperlink se puede ver abajo.

Ir a pagina de link

form - Forms

Sirven para recibir entradas del usuario.

El tag textarea es un cuadro de texto. "placeholder" Pone un texto por default que desaparece cuando se agrega texto, como el "Introducir Texto" de abajo.

El tag button es un boton

El tag input sirve para meter texto como entrada. El parametro "type" valida que esta entrada sea de algun tipo. El espacio de abajo requiere de un email.

footer - Pie de página

Utilizado para ponerle un pie de página al sitio. Este texto y su subtítulo son pies de página.
